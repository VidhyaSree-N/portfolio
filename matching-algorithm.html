Student-Faculty Matching Algorithm

This project focuses on automating the matching process between students and faculty members using a modified Gale-Shapley algorithm. The algorithm accounts for faculty capacities, handles ties in preference scores, and is optimized for scalability and accuracy.

The algorithm requires an Excel file with three sheets: Faculty Capacities, Faculty Preferences, and Student Preferences. It processes the data client-side for security and efficiency, producing accurate matches in seconds.

This project has been deployed successfully and has significantly reduced the manual workload of student-faculty pairing. The tool is live on GitHub Pages and available for demonstration.

Next Steps: Future enhancements include adding campus-specific matching and backup features to ensure robustness in diverse scenarios.
