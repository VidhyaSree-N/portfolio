The system was built using the MERN stack (MongoDB, Express.js, React.js, Node.js). As the team lead, I was responsible for stakeholder meetings, user requirement gathering, database schema design, and sprint planning using Agile methodologies.

One of the major challenges was integrating seamless gamification mechanics in a way that complemented academic objectives. We also ensured secure and scalable back-end architecture using JWT-based user authentication and modular APIs.

Outcome: The application streamlined participation tracking for instructors and made the process more transparent and motivating for students, fostering greater accountability and involvement.
